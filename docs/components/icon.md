# icon

> icon 作为 WheelHub 最简单的组件，这样推荐使用 iconfont SVG

[iconfont 官网](https://www.iconfont.cn/)

### 全局引用

下载 `iconfont.js` 后，需要在 `main.js`（或是其他入口 js 文件）引用

```
import "../iconfont.js";
```

### 代码

```
<g-icon name='iconName'></g-icon>
// 更换 'iconName' 即可
```

<ClientOnly><icon/></ClientOnly>
